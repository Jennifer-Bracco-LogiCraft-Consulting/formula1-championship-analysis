# Formula 1 Championship Analysis (2008 Season)

This project focuses on analyzing the results of the 2008 Formula 1 World Championship using Python.

## Objective
Transform race results into structured information to evaluate driver performance and generate final standings for both drivers and constructors.

## Data
The dataset contains race results from the 2008 Formula 1 season, including drivers, teams, race locations and finishing positions.

## Tools
- Python (base programming, no external libraries)
- Google Colab

## Results
- Individual driver performance (total points, wins, podiums)
- Final driver standings (exported as a .txt file)
- Constructor standings based on aggregated team points

## Notebook
The full analysis is available in the notebook file.
